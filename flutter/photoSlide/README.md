## 개발환경
- Flutter 3.10.6
- Dart 3.10
<br>

## 기능
- 좌우 스와이프로 이미지 변경 가능
- 특정 시간 지나면 자동으로 롤링
- 이미지 전체화면
- 상태바 색상 변경
<br>

## 주요 구성요소
- Image
- PageView
- Timer
- SystemChrome
- StatefulWidget
