## 개발환경
- Flutter 3.10.6  
- Dart 3.10
<br>

## 기능
- Google Maps API 활용해 지도 UI 구현
- 현재 위치 보여주기
- [현재 위치로 이동] 버튼 활용해 GPS상의 현재 위치로 이동
- 출근 가능 위치에서 [출근하기] 버튼 눌러 출석 체크 가능
- 출근 가능 위치 아니면 [출근하기] 버튼 보이지 않음  
<br>

## 주요 구성요소
- Google Maps
- Geolocator
- Dialog  
<br>
