# DessertoMe
---
<br>

## 개발환경
- Flutter 3.10.6  
- Dart 3.10
<br>

## 기능
- Riverpod 상태관리
- Pagination
- JWT 토큰관리 및 인증
- Optimistic Response
- JsonSerializable
- GoRouter
- Retrofit
- Dio 토큰관리 자동화
- Debounce and Throttle
<br>
