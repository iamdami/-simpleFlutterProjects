## 개발환경
- Flutter 3.10.6  
- Dart 3.10
<br>

## 기능
- 실제 핸드폰 이용해 1:1 영상채팅 가능
- 영상통화 종료
<br>

## 주요 구성요소
- Agora API
- 권한 관리
- WebRTC
<br>

## 플러그인
